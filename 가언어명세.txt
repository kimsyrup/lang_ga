--------------------------

   사용:   가=1   가=나  가=1+4
배열 [Yes]
   선언:   가(수)[100]
   사용:   가[2]
함수 [Yes]

    만약 가=2 {    }
    아니고만약 가=4 {     }
    아니면{    }

    반복 가=1~10(+2){     }
